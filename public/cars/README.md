# Car Images

Organize car images by slug in subdirectories:

```
cars/
├─ jeep-wrangler-sahara-4xe-2022-sky-top/
│  ├─ 1.jpg (main image)
│  ├─ 2.jpg (optional)
│  └─ 3.jpg (optional)
├─ audi-a5-cabrio-2022/
│  ├─ 1.jpg
│  └─ ...
```

## Requirements

- Format: JPG or WebP
- Size: Max 300-400KB per image
- Dimensions: Recommended 1200x800 or similar
- Optimized for web
- Naming: Use numbers (1.jpg, 2.jpg, etc.)

## Car Slugs

See `src/data/cars.ts` for the complete list of car slugs.
