# Hero Image/Video

Place the hero asset here:

- `hero.jpg` - Hero image (recommended: 1920x1080, optimized < 400KB)
- OR `hero.mp4` - Hero video (with fallback image)

## Requirements

- Format: JPG (image) or MP4 (video)
- Size: Max 400KB for images
- Dimensions: 1920x1080 or similar aspect ratio
- Optimized for web
